# Booking Barber

Aplikasi reservasi online untuk barber shop built with Laravel. Memudahkan pelanggan memesan layanan, dan admin mengelola layanan serta booking.

## Fitur
- Manajemen layanan (nama, harga, durasi)
- Form booking pelanggan (pilih layanan, tanggal, waktu)
- Dashboard admin untuk melihat & mengubah status booking
- Seeder contoh data layanan
- Built with Laravel, Vite (JS/CSS), and MySQL

## Tech Stack
- PHP 8+
- Laravel
- MySQL / MariaDB
- Composer
- Node.js & npm
- Vite (assets)

## Persyaratan
- PHP 8.1+
- Composer
- Node.js 16+
- MySQL atau MariaDB
- Git
